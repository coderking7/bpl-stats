# bpl-stats
Command-line tool for live scores and stats of Barclays Premier League

### Current Functionality
- Get Data Related to Points Table
- Next Three Fixtures of Every Team
- Upcoming Fixtures
- Recent Results
- Get Latest News
- Get List of Top Goalscorers
